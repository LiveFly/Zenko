.. _`|product| UI`:

Using the |product| UI
======================

The |product| UI is an application-based graphical user interface for the |product|
platform. While |product| offers its full functionality through a rich API, many
of its features are also available through this graphical user interface,
providing ease of use and excellent opportunities for visualizing managed data.
